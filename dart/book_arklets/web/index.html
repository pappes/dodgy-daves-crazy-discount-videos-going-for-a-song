Bookmarklet: rotate an image 1.0
Description:
when viewing a single image (not a HTML page) this bookmarklet wrapps the image location in HTML which allows customisation of:
  • height (and implicitly width)
  • top(y) and left(x) position
  • angle of rotation - from 0 to 360
